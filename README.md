# java-client

This is the java SDK for NoetherLab service

## Installation

    <dependency>
        <groupId>com.noetherlab</groupId>
        <artifactId>java-client</artifactId>
        <version>1.0.308</version>
    </dependency>

## Run unit test

You have to configure an env variable

    NOETHERLAB_API_KEY
